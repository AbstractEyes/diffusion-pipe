# Qwen DeepFashion (Real + Synthetic Full-Body Outfits)

A dataset of **160,015 fully synthetic (AI-generated) full-body fashion images** produced with
**Qwen-Image** + the **Qwen-Image-Lightning** 4-step LoRA. Outfit descriptions come from two
sources — the **real DeepFashion** caption set and a **synthetic outfit generator** — and a shared
prompt-augmentation policy (`fashion-v1`) renders them as **head-to-toe outfit photographs** with
**diversified wearers, backgrounds, poses, and framing** while preserving the **gender-appropriate
garment** of each source caption. It is intended as a broad, balanced pretraining substrate for
full-body fashion generation, complementing the portrait-oriented sister set
[`AbstractPhil/qwen-synth-characters`](https://huggingface.co/datasets/AbstractPhil/qwen-synth-characters).

> [!IMPORTANT]
> **These are not real people.** Every image is generated by a diffusion model from a text prompt;
> any resemblance to a real individual is coincidental. The demographic columns (`race`, `gender`,
> `age_band`, `hair`, …) record the **generation *intent*** — the attributes written into the
> prompt — **not measured/verified attributes of the rendered image.** Do not treat these columns as
> ground-truth annotations. The face-oriented columns `eye`, `expression`, `makeup`, and `jewelry`
> are **empty by design** here — the fashion policy injects only full-body attributes (it sets
> `hair` ~70% of the time and leaves the portrait-only fields blank).

## What's in it

- **160,015 images** at 1024-region resolutions: `1024×1024` (35.3%), `832×1216` (33.4%), `1216×832` (31.2%).
- Two provenance groups (see the `id` prefix and `source_prompt`):
  - **Real DeepFashion captions** (`id` prefix `deepfashion_`, ~12,015 base captions): the
    `deepfashion_caption` field of
    [`AbstractPhil/diffusion-pretrain-set-ft1`](https://huggingface.co/datasets/AbstractPhil/diffusion-pretrain-set-ft1)
    (config `deepfashion`, column `captions_source_json`). **Text only — no DeepFashion images are
    used or redistributed.** These captions are female-predominant and describe real garments.
  - **Synthetic outfits** (`id` prefix `fsyn_female_` / `fsyn_male_`): a gender-gated combinatorial
    outfit generator (garment × color/pattern × footwear × accessory) that **up-weights garment
    categories DeepFashion under-represents** (dresses, formalwear, outerwear, knitwear, occasion).
    Generated **predominantly female (~7:3)** with a parallel **male substrate** of similar outfits,
    so the set carries a gender-balanced relational structure without losing gender-appropriate
    garments.

The `image` column is a standard 🤗 `datasets` **Image** feature (PNG), so the Hub viewer renders
thumbnails and `load_dataset` decodes it directly.

## How it was generated

- **Models:** [`Qwen/Qwen-Image`](https://huggingface.co/Qwen/Qwen-Image) (~20B, Apache-2.0) +
  [`lightx2v/Qwen-Image-Lightning`](https://huggingface.co/lightx2v/Qwen-Image-Lightning)
  `Qwen-Image-Lightning-4steps-V1.0.safetensors`.
- **Inference:** 4 steps, `true_cfg_scale=1.0`, bf16, resident on 96 GB GPUs, 2-GPU sharded.
- **Prompt policy (`policy_version = fashion-v1`)** rewrites each outfit caption for full-body
  fashion:
  - **Outfit preserved, composition diversified** — the source garment (and its gender) is kept;
    the policy strips any "white background" phrasing and composes the scene: background
    (~18% studio / ~82% real-world settings), **full-length head-to-toe framing on every image**,
    plus pose (~75%), subject view (~55%), camera angle (~50%), and lighting (~60%).
  - **Demographic balancing** — when a caption does not state race/ethnicity, a label is sampled
    from a fairly-uniform distribution (with a deliberately-rare long tail flagged by `is_tail`) to
    counter the base model's default appearance; explicit source attributes are respected. Gender is
    detected from the source and **preserved**.
  - **Adult constraint** — age terms are stripped and an adult **25–35** band is enforced; juvenile
    nouns are de-minored (`girl→woman`, `boy→man`). See the age caveat below.
  - **Quality tier** — ~90% realistic color photography, ~10% deliberately "amateur" (slightly soft /
    grainy) for structural variety (`is_amateur`).
  - **Near-duplicate suppression** — a dependency-free MinHash/LSH token-shingle guard
    (`DiversityGuard`, threshold 0.82) resamples attributes when a prompt is too similar to an
    earlier one **within each generation process**.
  - Generation is **deterministic per `id`** (the `seed` column), so any row is reproducible.

## Dataset structure

| column | type | meaning |
|---|---|---|
| `id` | string | stable key; `deepfashion_{n}` = real-caption-derived, `fsyn_{gender}_b{batch}_{n}` = synthetic outfit |
| `image` | Image | the generated PNG (full-body) |
| `image_width`, `image_height` | int32 | pixel dimensions |
| `prompt` | string | the **augmented** prompt actually used for generation |
| `source_prompt` | string | the original outfit caption before augmentation |
| `race` | string | **intended** race/ethnicity label (generation intent, not verified) |
| `race_injected` | bool | whether the race was sampled (true) or taken from the source (false) |
| `is_tail` | bool | race is from the deliberately-rare "long tail" set |
| `gender` | string | intended subject (preserved from the source caption): `woman` / `man` / `person` |
| `age_band` | string | always `25-35` (prompt-level constraint; see caveat below) |
| `hair` | string | intended hair attribute (injected ~70% of rows; empty otherwise) |
| `eye`, `expression`, `makeup`, `jewelry` | string | **empty by design** (portrait-only fields, unused in the fashion policy) |
| `is_amateur` | bool | rendered with a lower-quality "amateur" style (~10%) |
| `seed` | int64 | deterministic per-`id` seed |
| `width_ratio` | string | e.g. `832x1216` |
| `policy_version` | string | augmentation policy version (`fashion-v1`) |

> Background (studio vs. real) and garment category are **embedded in the prompt text**, not stored
> as columns; they can be recovered with the dataset's own vocab (`qwen_extraction/fashion_vocab.py`),
> which is exactly how the composition numbers below were measured.

## Composition (actual, measured over all 160,015 rows)

- **Provenance:** synthetic female 101,000 (63.1%), synthetic male 47,000 (29.4%), real DeepFashion 12,015 (7.5%).
- **Gender (intended, preserved from source):** woman 112,785 (70.5%), man 47,018 (29.4%), unlabeled/person 212 (0.1%).
- **Race (intended):** caucasian 14.9%, then south_asian / middle_eastern / native_american / black /
  east_asian / hispanic_latino / southeast_asian ~8.0% each, pacific_islander 7.1%, multiracial 7.0%;
  deliberately-rare long tail (central_asian, mestizo, polynesian, scandinavian, mediterranean,
  persian, ethiopian) ~2.0% each. Long-tail fraction ~15%; 99.9% of labels were sampled (the source
  captions rarely state ethnicity), so this reflects the balanced sampler.
- **Background:** real-world settings 82.0%, studio 17.9% (text-derived; the policy appends one
  setting phrase verbatim).
- **Framing:** every row carries a full-length / head-to-toe framing cue.
- **Resolution:** `1024×1024` (35.3%), `832×1216` (33.4%), `1216×832` (31.2%).
- **Quality:** amateur-tier ~10.0%.
- **Synthetic garment mix (148,000 synthetic rows, 100% matched to the controlled vocab):** women's
  garments up-weight the categories DeepFashion under-represents — dresses 20,259, outerwear 17,108,
  knitwear 14,032, formal 13,956, athletic 11,499, occasion 10,165, bottoms 9,672, tops 9,138;
  men's — tops 8,805, formal 7,220, bottoms 7,022, outerwear 6,713, occasion 4,468, athletic 4,086,
  knitwear 3,857. (Real `deepfashion_` rows are free-text and intentionally fall outside this vocab.)
- **Duplication:** exact-duplicate prompt rate 0.005% (8 rows; the single most-repeated prompt occurs
  twice), normalized-duplicate rate 0.006% (9 rows); a `DiversityGuard` near-dup probe over a
  20,000-prompt sample found 0.005%. An image-saturation spot-check over 455 decoded images found
  **0 near-greyscale** (mean saturation 87/255, min 10.6) — **no monochrome images**.

## Intended uses

Research and development of generative image models, specifically **full-body / fashion** generation:
pretraining and fine-tuning data, data augmentation, garment- and pose-conditioned generation, and
studying demographic balance in synthetic fashion data.

## ⚠️ Limitations, biases, and required pre-processing

- **Age / minors — filter before use.** Prompts are constrained to adults (25–35) and juvenile nouns
  are de-minored, but **no automated age verification has been applied**, and a diffusion model can
  still render younger-looking subjects. **A strong age-verification pass and removal of any
  minor-appearing samples (and child-related tags) is a planned, not-yet-applied pre-use step** — the
  dataset is *not* age-filtered as published. No real minors are depicted; every image is synthetic
  from text. Run that filter before any downstream training.
- **Labels ≠ ground truth.** The demographic/attribute columns are prompt *intent*, not verified
  output attributes; do not use them to train or evaluate demographic classifiers or
  facial/garment-recognition systems for real people.
- **Background / garment stats are text-derived approximations.** Studio-vs-real is near-exact (the
  policy appends a verbatim phrase); garment-category matching is reliable for the **synthetic** rows
  only — the real `deepfashion_` captions are free-text and mostly fall outside the controlled vocab.
- **"Black and white" in a caption usually means the garment, not the photo.** Some real DeepFashion
  captions describe two-tone garments; these are **not** monochrome images (the saturation spot-check
  confirms no greyscale renders).
- **Female-predominant by design** (real DeepFashion is female-heavy and the synthetic mix is ~7:3
  female); the male substrate exists to provide parallel outfits, not parity.
- **Synthetic artifacts.** Expect occasional diffusion artifacts (hands, accessories, text, footwear).
  ~10% of images are intentionally lower-quality ("amateur").
- **Not for identifying or impersonating real people.**

DeepFashion *text* captions in `AbstractPhil/diffusion-pretrain-set-ft1` and from a synthetic outfit
